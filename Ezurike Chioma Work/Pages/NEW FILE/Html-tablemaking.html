BUILDING HTML TABLE

Making an HTML table,

To start, we will first need a table tag with a border inline element.

Everything else in the table will be inside this tag.


Inside the table we will have table rows, table cells, and table headers for the column headers.

Then we will add in the first table row, using the tr tag which means table-roll.


Inside this row we will want to put the column headers. We can do this by using the — table header — tags. let table th be

  • S/NO
  • NAME
  • AGE

By default the table headers are bold text and also centered within the cell.

Then we have our second row and inside, we have td tag since these are not headers it will not be bold. For the table data, to build out this table just add in numbers. Alright, lets say our S/NO IS 1, then name is Micheal, and Age is 27. That is td is

  • 1
  • Micheal
  • 27
I am sure there are other categories that I am forgetting, but we are just keeping it simple here.


Then in the next row, we will have tr tag, and td inside and we will add 2, Peter for name and 5 for Age. That is

  • 2
  • Peter
  • 5

Then in the next row, we will have tr tag, and td inside. td will be

  • 3
  • Joy
  • 23

Next row will be tr tag, td inside the tag and td will be

  • 4
  • Mary
  • 30


Our table will be

S/NO NAME AGE
1 Micheal 27
2 Peter 5
3 Joy 23
4 Mary 30
